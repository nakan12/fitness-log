# fitness-log

28日間の食事 × 運動プランを管理する1ファイル静的サイト。GitHub Pages 等にそのまま置ける。

## できること

- 開始日を任意に設定し、Week 1〜4 / DAY 1〜28 を動的に表示
- 食事記録（朝・昼・夜・間食・運動・メモ）／体重・ウエスト記録（簡易グラフ付）
- 連続記録（ストリーク）と進捗バーで継続を見える化
- 各日の予定を `.ics` でカレンダーアプリにエクスポート
- データはすべて端末の localStorage に保存（外部送信なし）

## 使い方

- `index.html` をブラウザで開く（GitHub Pages にデプロイすればモバイルから常時アクセス可）
- 右上の歯車アイコンから開始日を変更／「今日からスタート」／全データ削除
- 開始日を変更しても食事・体重の記録は日付タグで保持される

## ローカルで開く

```sh
python3 -m http.server 8080
# → http://localhost:8080/
```

## データキー（localStorage）

| キー | 内容 |
|---|---|
| `fl_settings_v1` | `{ startDate: "YYYY-MM-DD" }` |
| `fl_meal_log_v1` | 食事記録（日付ベース） |
| `fl_body_log_v1` | 体重・ウエスト記録（日付ベース） |

## 構成

1 ファイル構成。`index.html` のみ。外部依存は Google Fonts のみ。
